# FlutterPod4VSCodium

## features:
- onecommand installer
- developing for a physical android device

## planed:
- [ ] linux development
- [ ] web dev
## installation
clone the repo
```
git clone
```
Then just run the devEnv script and you should be good to go

```
chmod -R +x flutterpod4vsc
cd flutterpod4vsc
./devEnv.sh
```
